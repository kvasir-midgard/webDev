Boolean Logic
Everything starts with the idea that a statement is either True or False 
Then we can combine those initial statements to create more complex 
statements that also evaluate to True or False

-------------------------------------------------------------------------------
Equality Operators
== vs. ===
-------------------------------------------------------------------------------

var x = 99;

x == "99"  //true

x === "99" //false

var y = null;

y == undefined //true

y === undefined //false

"==" performs type coercion, while "===" does not

-------------------------------------------------------------------------------
A Few Interesting Cases
-------------------------------------------------------------------------------

true == "1"          //true

0 == false           //true

null == undefined    //true

NaN == NaN           //false

-------------------------------------------------------------------------------
Logical Operators
AND, OR, and NOT
-------------------------------------------------------------------------------

Assuming x = 5 and y = 9

Operator	Name	Example	            Result
&&	        AND	    x < 10 && x !== 5	false
||	        OR	    y > 9 || x === 5	true
!	        NOT	    !(x === y)	        true
